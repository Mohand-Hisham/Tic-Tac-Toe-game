# 🎮 Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built with **React.js**.  
Players can take turns, and the game announces a winner or draw.

## 🚀 Features
- Play vs friend
- Highlights the winner
- Reset game option

## 🛠️ Tech Stack
- React.js
- Html
- CSS 

## ▶️ Run locally
```bash
git clone https://github.com/yourusername/tic-tac-toe
cd tic-tac-toe
npm install
npm run dev
